## Getting Started

### Dependencies

- `npm`
- `node`

### Installing

- Clone the repository and install dependencies:

```
git clone https://github.com/Felipe-Navas/react-3d-vite-ts.git && cd react-3d-vite-ts && npm install
```

### Executing program

```
npm start
```
